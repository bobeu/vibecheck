const fetchVibrancyReport = async (tokenId: string): Promise<Report> => {

    const apiUrl = `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.5-flash-preview-09-2025:generateContent?key=${apiKey}`;

    const payload = {
        contents: [{ parts: [{ text: userQuery }] }],
        tools: [{ "google_search": {} }],

    const MAX_RETRIES = 5;

    const executeFetch = async (): Promise<Report> => {

                    method: 'POST',
                    headers: { 'Content-Type': 'application/json' },
                    body: JSON.stringify(payload)
                });

                if (!response.ok) {
                    if (response.status === 429 && attempt < MAX_RETRIES - 1) {
                        await new Promise(resolve => setTimeout(resolve, delay));
                        delay *= 2;
                        attempt++;
                        continue;

                const result = await response.json();

                if (candidate && candidate.content?.parts?.[0]?.text) {
                    const text = candidate.content.parts[0].text;
                    let sources: { uri: string; title: string; }[] = [];
                    const groundingMetadata = candidate.groundingMetadata;

                    if (groundingMetadata && groundingMetadata.groundingAttributions) {
                        sources = groundingMetadata.groundingAttributions

                                uri: attribution.web?.uri,
                                title: attribution.web?.title,
                            }))
                            .filter((source: any) => source.uri && source.title);
                    }

                    const mockBaseScore = TOKENS.find(t => t.id === tokenId)?.score || 70;
                    
                    return {
                        id: tokenId,
                        score: mockBaseScore,
                        details: {
                            codeHealth: Math.min(100, mockBaseScore + Math.floor(Math.random() * 10 - 5)),
                            communitySentiment: Math.min(100, mockBaseScore + Math.floor(Math.random() * 10 - 5)),
                            tokenomics: Math.min(100, mockBaseScore + Math.floor(Math.random() * 10 - 5)),
                            analysis: text,
                            sources: sources,
                        }
                    } as Report; // Type assertion since purchase info is added later
                } else {
                    throw new Error("Invalid response structure from the API.");
                }
            } catch (error) {
                attempt++;
                if (attempt === MAX_RETRIES) {
                    console.error("All retries failed:", error);
                    const mockBaseScore = TOKENS.find(t => t.id === tokenId)?.score || 70;
                    return {

The existing mock useEffect must be replaced with a real-time listener.

Target: The useEffect labeled 2. Real-time Report Purchase Listener.

Goal: Check if the user has purchased the report for the selectedToken.

Logic:

Define the report document path: /artifacts/{appId}/users/{userId}/purchasedReports/{selectedToken}.

Use onSnapshot on this document.

In the snapshot callback:

Use setPurchaseData(docSnap.data()) to store the purchase data (which contains purchased, purchaseTimestamp, and lastUpdateTimestamp).

If the document exists (docSnap.exists()) AND contains the field purchased: true, set setReportStatus('unlocked').

Otherwise, set setReportStatus('pending').

Return the unsubscribe cleanup function.

2. üí∞ Firestore Write: Initial Purchase (handleUnlockReport)
This function handles the $1.00 transaction for first-time access.

Target: The handleUnlockReport function.

Goal: Record the initial purchase data after successful payment.

Get the document reference (same path as step 1).

    3.  **Crucial:** Do **NOT** manually set `setReportStatus('unlocked')`. The `onSnapshot` listener (Step 1) will automatically detect this write and update the state for you.

3. üîÑ Firestore Write: Report Update (handleUpdateReport)
This new function handles the discounted $0.50 transaction to refresh the AI analysis.

Target: The handleUpdateReport function.

Goal: Update only the lastUpdateTimestamp to record the refresh and trigger a new AI fetch.

Logic (inside the if (txResult.success) block):

    3.  After the Firestore write is complete, manually trigger the report refresh by calling **`fetchReport(selectedToken)`** to immediately fetch the new AI analysis. The purchase status is already `unlocked`.
